#   Clinic Management System (C Language)

A **menu-driven, file-based Clinic Management System** developed in **C**.  
This project demonstrates the use of **structures, file handling, and modular programming** to manage clinic operations efficiently.

##  Features

### Patient Management
- Add new patient records
- View all patients
- Search patient by ID
- Update patient details
- Delete patient records

### Doctor Management
- Add new doctors
- View all registered doctors

### Appointment Management
- Book appointments
- View all appointments

---

##  Technologies Used
- **Programming Language:** C
- **Concepts Applied:**
  - Structures
  - File Handling (`fopen`, `fprintf`, `fscanf`)
  - Functions & Modular Programming
  - Conditional Statements & Loops
- **Compiler:** GCC / Dev-C++
- **Platform:** Windows
